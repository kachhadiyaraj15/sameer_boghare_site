## Content Management

All content is stored in separate directories for each language:

- **English**: `content/en/documents.md`
- **Hindi**: `content/hi/documents.md`
- **Gujarati**: `content/gu/documents.md`

To update content:
1. Navigate to the appropriate language folder
2. Edit the `documents.md` file
3. Update the corresponding data in `js/app.js`
